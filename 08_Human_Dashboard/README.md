# Human Dashboard

A **static, offline overview** of MoonBridge for first-time orientation and day-to-day navigation.

## Contents

| File | Purpose |
|------|---------|
| [dashboard.html](dashboard.html) | Local HTML overview — open in any browser, works offline |
| [dashboard_data_example.json](dashboard_data_example.json) | Sample dashboard data only (not live) |
| [Dashboard_Guide.md](Dashboard_Guide.md) | How to use the dashboard safely |

## Important

The dashboard **does not**:

- Scan your computer
- Read file contents
- Upload anything
- Run automation
- Connect to the network

It is a **human-readable map** with links to local markdown documentation.

## Quick start

1. Open `dashboard.html` in your file manager or browser
2. Use the cards to jump to relevant docs
3. Follow [Human_First_Workflow.md](../10_Automation_Plans/Human_First_Workflow.md) for real work
