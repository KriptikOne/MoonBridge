# MoonPool

**The controlled user-content intake zone for MoonBridge.**

Your real files enter the system here — only when you manually copy them in. Nothing outside MoonPool is auto-scanned.

## Standard substructure (categories 01–13)

Each category contains:

- `Approved_Batches/` — cleansed material (child paths must be allowlisted to scan)
- `Working/` — active organisation
- `Finished/` — ready to use
- `Archive/` — kept but inactive

See [How This System Works](../00_Read_Me_First/How_This_System_Works.md) for the batch lifecycle.
