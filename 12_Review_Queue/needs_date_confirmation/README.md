# Needs Date Confirmation

Place records here when a **plausible date exists** but still needs **human confirmation** before handover or Approved Files status.

---

## Typical signals

- `dateConfidence: medium` or `high` but no `verifiedDate`
- Filename date accepted tentatively for planning only

---

## Manual next step

1. Review all chronology fields and notes
2. Set `verifiedDate` and `dateConfidence: verified` when confirmed
3. Remove from queue in your index when complete

*Suggested dates are not verified dates.*
